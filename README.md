# lean-logic-dsl

A lightweight domain-specific language (DSL) for propositional logic in Lean 4, built to explore formal syntax, semantic evaluation, pretty-printing, and metaprogramming foundations for future extensions into modal and epistemic logic.

This project is designed as a **clean, extensible formal language toolkit**.The long-term goal is to provide a structured base for experimenting with:

- propositional logic
- modal logic
- epistemic logic
- algebraic simplification
